# Projeto AirAnC

Frontend construido com ReactJS utilizando React Hooks.

## Tecnologias Utilizadas

* ReactJS
* Yarn
* Axios
* Socket.io

## Instalação

```bash
# Para instalar as dependencias do projeto
yarn install
# Executar Projeto
yarn start
```
